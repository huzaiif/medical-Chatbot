# Medical Chatbot

This is a Multiple Disease Prediction System with an integrated Health Bot.

## Features
- **Health Bot**: A chat interface powered by Gemini to answer health-related queries.
- **Diabetes Prediction**: Predicts diabetes risk based on user inputs.
- **Heart Disease Prediction**: Predicts heart disease risk.
- **Parkinson's Prediction**: Predicts Parkinson's disease risk.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Create a `.env` file with `GOOGLE_API_KEY`.
4. Run the app: `streamlit run app.py`.
